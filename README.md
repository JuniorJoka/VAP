# VAP

Modern media player for modern people

## Run Locally

Clone the project

```bash
  git clone https://github.com/JuniorJoka/VAP.git
```

Go to the project directory

```bash
  cd VAP
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
